Loan Application Prediction

  • Github URL: Project Link

Project Overview and Objective: Conducted a comprehensive analysis on a loan prediction dataset to understand factors influencing loan approval decisions and predicted the probability of each applicant receiving a loan, presenting results in a new table.

Data Preparation and EDA: Cleaned and preprocessed the dataset, performed exploratory data analysis (EDA) using descriptive statistics and visualizations to identify key trends and patterns affecting loan approval.

Factor Analysis and Predictive Modeling: Analyzed the impact of various factors (e.g., income, loan amount, credit history) using statistical methods, and developed logistic regression models to predict loan approval probabilities based on these factors.

Model Validation and Probability Calculation: Validated predictive models using cross-validation and ROC curve analysis to ensure accuracy, and calculated loan approval probabilities for each applicant, displaying these in a new table.

SAS Programming and Skill Application: Utilized SAS for data analysis, modeling, and visualization. Improved proficiency in SAS programming, data cleaning, exploratory analysis, predictive modeling, and result visualization. Enhanced problem-solving skills and ability to interpret complex data for data-driven decisions.
