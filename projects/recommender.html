Social Media Website

Project Overview: Designed and developed an interactive social media platform inspired by Twitter, allowing users to post messages up to 500 words and start discussion threads, fostering user engagement and conversation.

Technology Stack: Utilized Java for backend development with Spring Boot, HTML, CSS, and Bootstrap for frontend design, and MySQL for database management, ensuring a robust and visually appealing site

APIs and Security: Developed RESTful APIs in Java for seamless frontend-backend communication and implemented secure user authentication and authorization with Spring Security to protect user data and interactions.

Features and Scalability: Added features like liking, commenting, and sharing posts to enhance interaction. Designed the architecture for scalability, ensuring the site can handle growing numbers of users and posts efficiently.

Testing and Skill Development: Conducted thorough testing using JUnit and Selenium to ensure application reliability. Gained experience in full-stack development, improving skills in Java programming, web development, real-time data handling, and security best practices.
